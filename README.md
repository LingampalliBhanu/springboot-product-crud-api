# Spring Boot Product CRUD API

This project is a REST API built using Spring Boot for managing products.

## Technologies Used

- Java
- Spring Boot
- Spring Data JPA
- MySQL
- Maven

## Features

- Add Product
- View Products
- Get Product by ID
- Delete Product

## API Endpoints

GET /products  
POST /products  
GET /products/{id}  
DELETE /products/{id}
